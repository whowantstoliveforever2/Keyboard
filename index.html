Раскладка для памирских языков
Бета-версия

Что это такое?

Pamiri Keyboard — это клавиатура для устройств Android, созданная специально для письма на памирских языках. Раскладка основана на алфавите, предложенном выдающимся памирологом Додихудо Карамшоевым в его трёхтомном шугнанско-русском словаре.

ℹ️ Это бета-версия клавиатуры. Приложение активно разрабатывается и будет улучшаться: добавится автодополнение, исправятся ошибки, появятся новые функции. Ваша обратная связь очень важна для развития проекта!

Как установить?

  1. Скачайте файл pamiri-keyboard.apk
  2. Откройте файл и нажмите «Установить»
  3. Перейдите в Настройки → Язык и ввод → Виртуальная клавиатура
  4. Включите «Pamiri Keyboard»
  5. При наборе текста переключитесь на нашу клавиатуру
